## Local Development

To view this website locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/timjavins/timjavins.github.io.git
   ```

2. Navigate to the project directory:
   ```bash
   cd timjavins.github.io
   ```

3. Open `index.html` in your web browser:
   ```bash
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   
   # On Windows
   start index.html
   ```

Alternatively, you can use a local web server:

```bash
# Using Python 3
python -m http.server 8000

# Then visit http://localhost:8000
```

## Deployment

This site is automatically deployed via GitHub Pages. Any changes pushed to the main branch will be automatically published to the live site.

### GitHub Pages Configuration

- **Source**: Deploy from the root directory of the main branch
- **Custom Domain**: Not configured (using default `username.github.io` domain)
- **HTTPS**: Enabled by default
